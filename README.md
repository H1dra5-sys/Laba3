# Лабараторная работа: Змейка
## Автор
Коновалов Константин Михайлович
## Цель
Реализовать простейшую змейку.

## Описание
Пошаговая змейка, которая ничем не отличается от обычной, ешь еду - увеличивается хвост, врезаешься в хвост - смерть, управление реализовано на WASD, в этой версии границы экрана замкнуты.
