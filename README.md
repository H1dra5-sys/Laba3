# Лабараторная работа: Арабские и Римские цифры
## Автор
Коновалов Константин Михайлович
## Цель
Создать программу для перевода из арабских цифр в римские и наоборотв

## Описание
Программа переводит из римских в арабские и наоборот, нужно лишь написать число или сочетание букв, ограничение от 1 до 3999
